<h3>Este script tem como objetivo controlar um fita/módulo de LED RGB via Bluetooth, utilizando o microcontrolador ESP32.</h3>

<b>Materiais utilizados:</b>
<br>1 - Microcontrolador ESP32
<br>1 - Módulo LED RGB
<br>3 - Transistores BC547
<br>3 - Resistores 1K Ohm
<br>* - Jumpers e Protobord
<br>1 - Fonte 12 voltz
<br>1 - Fonte 5 voltz

<b>Aplicativo Android para controlar e comunicar com o microcontrolador:</b>
<br>Color LED Controller (https://play.google.com/store/apps/details?id=appinventor.ai_yuanryan_chen.BT_LED&hl=pt_BR)

Para mais detalhes segue o link do video tutorial: https://www.youtube.com/watch?v=B5cCDuziOh4

<h3>This script aims to control an RGB LED strip/module via Bluetooth, using the ESP32 microcontroller.</h3>

<b>Materials used:</b>
<br>1 - ESP32 microcontroller
<br>1 - RGB LED module
<br>3 - BC547 transistors
<br>3 - 1K Ohm resistors
<br>* - Jumpers and Protobord
<br>1 - 12 volt source
<br>1 - 5 volt source

<b>Android application to control and communicate with the microcontroller:</b>
<br>Color LED Controller (https://play.google.com/store/apps/details?id=appinventor.ai_yuanryan_chen.BT_LED)

For more details follow the video tutorial link: https://www.youtube.com/watch?v=B5cCDuziOh4
